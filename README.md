# Peachess

This project aims to develop a fully functional chess game in Python, using an OOP design that makes it possible to modify the behavior of traditional chess pieces, add new pieces, or even change the rules or the structure of the chessboard easily, following SOLID principles and design patterns.

The reason for doing this project is to learn Python and become familiar with the language. I already know C++ and TypeScript, among other languages, so I thought this project could be fun and ideal for that purpose.
